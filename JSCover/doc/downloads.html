ZIP Release

The latest release is v1.0.5.

This release contains two notable enhancements:

  • Using HTML5 localStorage instead of an iframe to hold coverage data between page loads
  • URI to File-System Path Translation for custom JavaScript loaders in proxy mode

See the release notes for more information.
